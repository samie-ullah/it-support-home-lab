# Windows Server RRAS VPN Lab

## Overview
This lab documents the implementation of a Remote Access VPN using Windows Server RRAS with Active Directory authentication.

## Environment
- Windows Server (RRAS + Active Directory)
- Windows 11 VPN client
- Dual-NIC configuration (Internal Network + NAT)
- VirtualBox

## Key Tasks
- Installed and configured Routing and Remote Access Services (RRAS)
- Implemented VPN and NAT using a dual-NIC architecture
- Enabled Active Directory-authenticated VPN access
- Connected Windows 11 client via VPN

## Troubleshooting
- Resolved VPN connectivity issues related to firewall profiles and GRE traffic
- Troubleshot VPN protocol negotiation and routing issues
- Identified NAT loopback limitations in a virtualised environment

## Skills Demonstrated
VPN, RRAS, Network Troubleshooting, Authentication, Routing
