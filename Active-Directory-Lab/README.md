# Active Directory Home Lab
## Overview
This lab documents the setup and administration of an Active Directory domain environment used to support user authentication and access control.

## Environment
- Windows Server (Domain Controller)
- Windows 11 (Domain-joined client)
- VirtualBox

## Key Tasks
- Installed and configured Active Directory Domain Services (AD DS) and DNS
- Joined Windows 11 client machines to the domain
- Created and managed domain users and security groups
- Performed password resets and account unlocks

## Troubleshooting
- Investigated account lockouts using Event Viewer (Event IDs 4740, 4625)
- Resolved Group Policy issues caused by time synchronisation and secure channel problems

## Skills Demonstrated
Active Directory, User Management, Group Policy, Authentication, Troubleshooting
