# IT Support Home Lab
Hands-on IT Support home lab covering Active Directory, VPN, File Server, Group Policy, and real-world troubleshooting scenarios.
## Overview
This repository documents a practical IT Support lab built using Windows Server and Windows 11 virtual machines.  
The lab simulates real service desk and junior system administrator tasks, focusing on user support, access control, networking, and troubleshooting.
## Technologies Used
- Windows Server
- Windows 11
- Active Directory Domain Services (AD DS)
- DNS
- Group Policy
- RRAS (VPN)
- NTFS Permissions
- VirtualBox
- ServiceNow (ticketing concepts)
## Lab Projects
- Active Directory Domain Setup & User Management
- Windows Server RRAS VPN (Remote Access)
- File Server & Permissions (NTFS & AD Groups)
- Group Policy Troubleshooting
- Network & Authentication Issue Resolution
## Troubleshooting Focus
This lab includes documented troubleshooting of:
- Group Policy failures caused by time synchronisation issues
- Account lockouts and authentication failures
- VPN connectivity issues (firewall, GRE traffic, NAT loopback)
- User access and permissions problems

## Purpose
This lab was created to strengthen hands-on IT Support skills and provide real examples of troubleshooting scenarios commonly encountered in service desk and junior IT roles.
