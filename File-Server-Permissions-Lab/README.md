# File Server & Permissions Lab

## Overview
This lab documents the creation and security of a centralised Windows file server integrated with Active Directory.

## Key Tasks
- Created shared department folders (HR, IT, Finance)
- Implemented NTFS permissions using Active Directory security groups
- Applied best practice access control (open share permissions, restrictive NTFS permissions)
- Automated drive mapping using Group Policy

## Troubleshooting
- Resolved access denied issues caused by incorrect permissions
- Fixed missing access due to incorrect group membership
- Refreshed Group Policy to apply permission changes

## Skills Demonstrated
File Server, NTFS Permissions, Active Directory Groups, Group Policy
